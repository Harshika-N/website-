***Web Design & Development Landing Page***
This repository contains the source code for a stylish, modern landing page designed for promoting a Web Design and Development course. It features a navigation bar, a clear call-to-action, and a prominent login form.

**Features**
*Gradient Background:* A pleasant background gradient (linear-gradient(120deg, #c5ccd0, #00b4d8)) for the entire page.

*Navigation:* Clean top navigation with links and a search bar (srch and btn).

*Content Focus:* Large, engaging title (Web Design & Development Course) with key emphasis on the word Development (highlighted in orangered).

*Login Form:* A translucent, fixed-position form for user authentication, styled with orangered accents.

*Responsive Base:* Structured for a 1200px width layout, ready for further responsiveness adjustments.

*Hover Effects:* Smooth color transitions on navigation links (ul li a:hover), the main content button (cn:hover), and the login button (btnn:hover).

 **Getting Started**
Since this is a static website, getting it running is extremely simple!

**Prerequisites**
You need only a modern web browser (Chrome, Firefox, Edge, Safari, etc.).
